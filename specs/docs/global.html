    roll(array) → {Object}

    Get a random fortune from the array of fortunes.

    Get a random fortune from the array of fortunes. Each fortune has the following properties:

    {
  "title": "Title",
  "description": "Sample description",
  "effects": {
    "rest": <Integer>,
    "happiness": <Integer>
    "skills": {
      "test_taking": <Integer>,
      "coding": <Integer>,
      "thinking": <Integer>
     }
  }
}

    Each integer is a value between -100 and 100 inclusive, and reflects the stat changes to the user. For example, a fortune with a rest value of 10 will increase the user's rest by 10.
