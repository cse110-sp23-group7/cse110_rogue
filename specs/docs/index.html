    Setup

    1. Install Node.js and npm (Node Package Manager):

    Linux:

    sudo apt install nodejs npm

    MacOS:

    brew install node

    Windows:

    winget install -e --id OpenJS.NodeJS
    1. Clone the repository:
    git clone https://github.com/cse110-sp23-group7/cse110_rogue.git
    1. Install dependencies:
    npm install

    Current dependencies are:

    • jest for unit testing
    • jsdoc for documentation generation

    Usage

    Run unit tests:

    npm run test

    Generate documentation:

    npm run docs

    Docs will be generated in the docs directory. and can be opened using a browser. These docs are generated using code comments, so be sure to comment your code!

    Launch website:

    • Open index.html in a browser. Note that index.html will automatically redirect the user to home.html.
